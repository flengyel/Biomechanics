# Biomechanics Diagnostic Tutor

Repository scaffold for a Python-first biomechanics diagnostic tutor.

The initial implementation should focus on a formal diagnostic core, worksheet-derived task fixtures, text demos, and unit tests before any graphical UI work.

## Initial Commands

```bash
pytest
python -m biomech_tutor.tasks.validate_all
python -m biomech_tutor.demo.elbow_biceps_curl_2d
python -m biomech_tutor.demo.knee_lower_leg_jump_dynamics
```
